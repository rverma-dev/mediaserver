# Weekend Watch Planner

Create practical plans that fit real time and mood.

## Workflow

1. Ask or infer mood (fun, intense, family, sci-fi, Hindi, binge series).
2. Build 3–6 title mix with runtime-aware sequencing.
3. Include one backup title per mood when possible.
4. Deliver in chat-native visual format with Seerr direct links.
5. Offer a tighter “Top 3 only” option.

## Planning Heuristics

- Mix heavy + light titles to avoid fatigue.
- Keep one short/easy option in every pack.
- Prefer complete mini-series when user wants a binge.

## Output

- Headline pack name.
- Poster-card style entries.
- Optional schedule suggestion (Fri/Sat/Sun split).
